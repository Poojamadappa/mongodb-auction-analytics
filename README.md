# MongoDB Auction Analytics Project

## Overview
This project analyzes 1,000+ auction records using MongoDB to demonstrate document-based data modeling and analytical querying using the Aggregation Framework.

## Dataset
- Auction data stored as JSON documents
- Includes embedded venue details, auctioneer profiles, and online bidder locations

## Key Features
- Advanced filtering using MongoDB query operators
- Aggregation pipelines using $match, $group, $unwind, $avg, $sum, $max, and $sort
- Analytical insights on bidder activity, venue performance, auction fees, and auction categories
- Query validation and visualization using MongoDB Compass
- Cloud deployment and querying via MongoDB Atlas

## Tools & Technologies
- MongoDB Shell
- MongoDB Aggregation Framework
- MongoDB Compass
- MongoDB Atlas

## How to Run
1. Import the dataset using mongoimport
2. Run queries using mongosh or MongoDB Compass
3. Review aggregation results in Compass or Atlas
